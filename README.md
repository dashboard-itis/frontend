# SecCur

**SecCur** — дашборд студента и личный секретарь куратора.

Веб-платформа для визуализации и анализа учебной динамики студентов.

## Технологический стек

* React
* CRA
* TypeScript

## Запуск проекта

Установить зависимости:

```
npm install
```

Запустить проект в режиме разработки:

```
npm start
```
